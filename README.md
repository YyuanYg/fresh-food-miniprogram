# 鲜食优选 - 净菜商城微信小程序

## 项目介绍

鲜食优选是一个专注于净菜配送的微信小程序，提供新鲜蔬菜、水果、肉禽蛋等食材的在线购买服务。本项目基于微信小程序原生开发，采用WXML、WXSS、JavaScript技术栈。

## 功能特点

- 清新自然的绿色主题设计，符合净菜健康新鲜的定位
- 响应式布局，完美适配各种尺寸的手机屏幕
- 轮播Banner展示促销活动
- 分类导航快速找到所需商品
- 限时特惠区域带有倒计时功能
- 新品推荐和猜你喜欢商品展示
- 完整的购物车功能，支持添加、删除、修改数量
- 精美的动画效果增强用户体验

## 如何使用

1. 下载并安装微信开发者工具
2. 克隆本项目到本地
3. 在微信开发者工具中导入项目
4. 配置AppID（可以使用测试号）
5. 运行项目进行开发和调试

## 开发说明

### 数据存储

- 购物车数据使用微信小程序的本地存储（wx.setStorageSync）
- 商品数据暂时硬编码在页面中，实际项目中应从服务器获取

### 主要功能模块

1. **首页轮播**：使用微信小程序的swiper组件实现
2. **分类导航**：使用flex布局实现网格导航
3. **限时特惠**：使用scroll-view实现横向滚动，包含倒计时功能
4. **商品展示**：使用grid布局实现商品网格展示
5. **购物车**：使用模态框实现购物车弹窗，支持商品数量调整

### 样式设计

- 采用绿色为主色调，符合生鲜食品的清新自然定位
- 使用阴影和圆角提升UI的现代感
- 添加适当的动画效果提升用户体验

## 注意事项

1. 本项目为演示版本，商品数据和图片链接均为示例
2. 实际开发中应连接真实的后端API获取数据
3. 购物车结算功能需要接入支付接口
4. 图片资源建议使用CDN加速
